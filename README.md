# Sistema de Consultas DGINCD

Carpeta preparada para publicar en GitHub Pages.

Incluye solo la app estatica:

- index.html
- app.js
- styles.css
- config.js

No subir datos personales, CSV, Excel, .env, SQL ni scripts de carga.

La seguridad de datos depende de Supabase Auth y de que las RPC esten habilitadas solo para usuarios authenticated.
